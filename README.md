# Twin-lapia

Twin-lapia is a digital twin platform for sustainable tilapia farming.  
Built on Dynamic Energy Budget (DEB) theory, it simulates fish growth, metabolism, and water-quality dynamics to optimize feeding, reduce waste, improve profitability, and lower the carbon footprint of aquaculture.

---

## Features

- Predictive feeding optimization  
- Real-time growth and metabolism simulation  
- Water-quality interaction modeling  
- Feed waste reduction  
- Carbon footprint estimation  
- Traceability-ready farm data
